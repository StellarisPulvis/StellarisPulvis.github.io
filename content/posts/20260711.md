如今多数智能手机的相机应用都提供「专业模式」（Pro Mode），让拍摄者像使用单反一样手动控制曝光与对焦。要运用好这个模式，首先要理解五个核心参数：**ISO、WB、Shutter、EV、Focus**。下面逐一介绍它们的含义、调节效果以及典型使用场景。

---

## ISO（感光度）

**全称**：International Organization for Standardization（国际标准化组织制定的感光标准）

**定义**：ISO 值决定了相机传感器对光线的敏感程度。

**效果**：

| ISO 值     | 传感器灵敏度 | 噪点水平           | 适用场景                     |
| ---------- | ------------ | ------------------ | ---------------------------- |
| 50 / 100   | 低           | 极低，画质纯净     | 白天强光、户外、三脚架长曝   |
| 200 / 400  | 中等         | 轻微噪点           | 阴天、室内灯光               |
| 800 / 1600 | 高           | 明显噪点           | 黄昏、暗光手持               |
| 3200+      | 极高         | 噪点显著，细节丢失 | 星空、极暗环境（需降噪处理） |

**使用要点**：

- **ISO 越低越好**——在保证安全快门的前提下，尽量压低 ISO 以获得最干净的画质。
- 高 ISO 虽然能提亮画面，但代价是**噪点增加、动态范围缩减**。
- 手机传感器比单反小得多，高 ISO 噪点更明显，因此在手机上更应优先降低 ISO。

---

## WB（白平衡 / White Balance）

**定义**：白平衡用于校正不同光源造成的偏色，使白色物体在不同光线下仍然呈现为白色。它的核心单位是**色温（K，开尔文）**。

**常见色温范围**：

| 色温值（K） | 对应光源     | 画面色调   |
| ----------- | ------------ | ---------- |
| 2000 - 3000 | 烛光、钨丝灯 | 偏黄（暖） |
| 4000 - 5000 | 日光灯、阴天 | 中性       |
| 5500 - 6500 | 正午阳光     | 标准日光   |
| 7000 - 9000 | 阴影、多云   | 偏蓝（冷） |

**实际调节规律**：

- **调低 K 值（如 3000K）** → 画面偏蓝，抵消暖色光
- **调高 K 值（如 7000K）** → 画面偏黄，抵消冷色光
- 手机专业模式通常提供预设（日光、阴天、荧光灯、钨丝灯）和手动 K 值调节。

**使用要点**：

- 星空摄影建议手动设到 **4500K - 5000K** 左右，避免自动白平衡在暗场中飘忽不定。
- 如果想拍出「更蓝」的夜空，可适当降低 K 值。

---

## Shutter（快门速度）

**定义**：快门速度是相机传感器从开始曝光到结束曝光所经历的时间，单位通常为秒（s）或分数秒（1/xxx s）。

**效果**：

| 快门速度        | 进光量 | 运动表现       | 适用场景         |
| --------------- | ------ | -------------- | ---------------- |
| 1/1000s 以上    | 极少   | 定格高速运动   | 飞鸟、赛车、体育 |
| 1/125s - 1/500s | 少     | 定格日常运动   | 人像、街拍       |
| 1/30s - 1/60s   | 中等   | 轻微手震风险   | 室内、阴天手持   |
| 1s - 30s        | 大量   | 运动物体变模糊 | 夜景、车流、星空 |

**安全快门**：为了手持拍摄不模糊，快门速度一般不应低于焦距的倒数。手机主摄多为 26mm（等效），安全快门约 **1/30s**。低于 1/15s 建议使用三脚架。

**使用要点**：

- 慢速快门下任何抖动都会被放大，**三脚架是必需品**。
- 拍摄星空通常需要 **15s - 30s** 的快门速度；超过 30s 可能出现星星拖尾（因地球自转）。
- 快门速度与 ISO 需要配合调节：快门变慢一档（进光量翻倍），ISO 可以降低一档。

---

## EV（曝光补偿 / Exposure Value）

**定义**：曝光补偿是在相机自动测光结果基础上，人为增减画面的亮度。单位是 **EV**，每增减 1 EV 代表亮度翻倍或减半。

**常见范围**：通常 -3 EV 到 +3 EV，以 1/3 档为步进。

**效果**：

| EV 值   | 画面亮度               | 适用场景                               |
| ------- | ---------------------- | -------------------------------------- |
| -1 ~ -3 | 变暗                   | 高光过曝场景（如雪地、逆光主体）       |
| 0       | 相机测光认为的标准亮度 | 多数常规场景                           |
| +1 ~ +3 | 变亮                   | 暗部偏黑场景（如大面积深色、逆光人像） |

**白加黑减原则**：拍摄雪景（大面积白色）相机容易测光偏暗，需要 **+EV**；拍摄黑色物体（大面积暗色）相机容易测光偏亮，需要 **-EV**。

**与 ISO/Shutter 的关系**：

- EV 在专业模式中通常通过改变快门速度或 ISO 来实现。
- 在手机上，如果同时手动设定了 ISO 和快门，EV 调节可能被锁定或不起作用。

---

## Focus（对焦）

**定义**：Focus（对焦）调节镜头与传感器之间的距离，使被摄主体成像清晰。专业模式提供**手动对焦（MF）**，由用户滑动滑块控制。

**常见对焦距离**：

| MF 滑块位置    | 合焦距离       | 适用场景         |
| -------------- | -------------- | ---------------- |
| 最近端（微距） | ~2cm - 10cm    | 花卉、昆虫、纹理 |
| 中段           | 几十厘米到几米 | 人像、美食       |
| 远端           | 几米到几十米   | 风景、建筑       |
| 无穷远（∞）    | 极远           | 星空、城市远景   |

**辅助功能**：

- **峰值对焦**：很多手机在 MF 模式下会对合焦物体的边缘显示彩色高亮（红色/蓝色），便于判断是否对焦准确。
- **放大对焦**：部分手机支持在 MF 时局部放大画面，用于精细对焦。

**使用要点**：

- 拍摄星空时将 MF 拉到**无穷远（∞）**，再往回微调一点点，直到星星呈现最锐利的点状。
- 拍微距时用 MF 更容易找到最佳焦平面，避免自动对焦反复犹豫。

---

以上就是在手机专业模式中五个核心参数的全部介绍。理解每个参数的作用和相互影响之后，面对任何拍摄场景都能更有把握地做出调整。

Most modern smartphones come with a **Pro Mode** in their camera app, allowing you to control exposure and focus manually just like a DSLR. To master this mode, you need to understand five core parameters: **ISO, WB, Shutter, EV, and Focus**. Below is a breakdown of each — what it means, how adjusting it affects your image, and typical use cases.

---

## ISO (Light Sensitivity)

**Full name**: International Organization for Standardization (the sensitivity standard defined by the ISO)

**Definition**: ISO determines how sensitive the camera sensor is to light.

**Effect**:

| ISO Value  | Sensor Sensitivity | Noise Level          | Typical Use Cases                  |
| ---------- | ----------------- | -------------------- | ---------------------------------- |
| 50 / 100   | Low               | Minimal, clean image | Bright daylight, outdoor, tripod long exposure |
| 200 / 400  | Medium            | Slight noise         | Overcast, indoor lighting          |
| 800 / 1600 | High              | Noticeable noise     | Dusk, low-light handheld           |
| 3200+      | Very High         | Heavy noise, detail loss | Starry sky, extreme darkness (needs denoising) |

**Tips**:

- **Lower ISO is always better** — keep ISO as low as possible while maintaining a safe shutter speed for the cleanest image quality.
- Higher ISO brightens the image but at the cost of **increased noise and reduced dynamic range**.
- Smartphone sensors are much smaller than DSLR sensors, so high-ISO noise is more pronounced. Prioritize keeping the ISO low.

---

## WB (White Balance)

**Definition**: White balance corrects color casts caused by different light sources, ensuring white objects appear white under any lighting. Its core unit is **color temperature (K, Kelvin)**.

**Common color temperature ranges**:

| Color Temp (K) | Light Source        | Tint               |
| -------------- | ------------------- | ------------------ |
| 2000 - 3000    | Candle, tungsten    | Warm (yellowish)   |
| 4000 - 5000    | Fluorescent, overcast | Neutral          |
| 5500 - 6500    | Noon sunlight       | Standard daylight  |
| 7000 - 9000    | Shade, cloudy       | Cool (bluish)      |

**Adjustment behavior**:

- **Lower K value (e.g., 3000K)** → image turns bluer, counteracting warm light
- **Higher K value (e.g., 7000K)** → image turns yellower, counteracting cool light
- Pro Mode usually offers presets (Daylight, Cloudy, Fluorescent, Tungsten) and manual K-value adjustment.

**Tips**:

- For astrophotography, set WB manually to **4500K - 5000K** to avoid auto white balance drifting in dark scenes.
- If you want a "bluer" night sky, lower the K value slightly.

---

## Shutter (Shutter Speed)

**Definition**: Shutter speed is the duration the sensor is exposed to light, measured in seconds (s) or fractions of a second (1/xxx s).

**Effect**:

| Shutter Speed    | Light Intake | Motion Rendering     | Typical Use Cases        |
| --------------- | ------------ | -------------------- | ------------------------ |
| 1/1000s+        | Minimal      | Freezes fast motion  | Birds, racing, sports    |
| 1/125s - 1/500s | Low          | Freezes everyday motion | Portraits, street     |
| 1/30s - 1/60s   | Medium       | Slight blur risk     | Indoor, overcast handheld |
| 1s - 30s        | Large        | Moving objects blur  | Nightscapes, light trails, stars |

**Safe shutter speed**: To avoid blur when shooting handheld, shutter speed should generally not be slower than the reciprocal of the focal length. Most phone main cameras are around 26mm (equivalent), giving a safe shutter speed of about **1/30s**. Use a tripod when slower than 1/15s.

**Tips**:

- Any movement is magnified at slow shutter speeds — **a tripod is essential**.
- Astrophotography usually needs **15s - 30s** exposures; going beyond 30s may cause star trails (due to Earth's rotation).
- Shutter speed and ISO work together: halving shutter speed (doubling light) lets you lower ISO by one stop.

---

## EV (Exposure Compensation)

**Definition**: Exposure compensation lets you manually brighten or darken the image on top of the camera's metered exposure. The unit is **EV**; ±1 EV doubles or halves the brightness.

**Common range**: Typically -3 EV to +3 EV in 1/3-stop increments.

**Effect**:

| EV Value  | Image Brightness     | Typical Use Cases                    |
| --------- | -------------------- | ------------------------------------ |
| -1 to -3  | Darker               | Highlights blowing out (snow, backlit subjects) |
| 0         | Metered normal       | Most everyday scenes                 |
| +1 to +3  | Brighter             | Dark areas too dark (large dark regions, backlit portraits) |

**"Whiter brighter, darker darker" rule**: Snowy scenes (large white areas) trick the meter into underexposing — use **+EV**. Dark subjects (large black areas) trick the meter into overexposing — use **-EV**.

**Relationship with ISO/Shutter**:

- In Pro Mode, EV is usually achieved by adjusting shutter speed or ISO.
- If both ISO and shutter are manually set, EV adjustment may be locked or have no effect.

---

## Focus

**Definition**: Focus adjusts the distance between the lens and the sensor to make the subject sharp. Pro Mode offers **manual focus (MF)** controlled by a slider.

**Common focus distances**:

| MF Slider Position | Focus Distance    | Typical Use Cases             |
| ----------------- | ----------------- | ----------------------------- |
| Closest (Macro)   | ~2cm - 10cm       | Flowers, insects, textures    |
| Mid-range         | Tens of cm to meters | Portraits, food            |
| Far               | Meters to tens of meters | Landscapes, architecture |
| Infinity (∞)      | Very far          | Stars, city skylines          |

**Auxiliary features**:

- **Focus peaking**: Many phones show colored highlights (red/blue) on in-focus edges when using MF, making it easy to confirm focus.
- **Focus magnification**: Some phones let you zoom in on a portion of the frame during MF for fine-tuning.

**Tips**:

- For star photography, set MF to **infinity (∞)**, then nudge it back slightly until stars appear as sharp points.
- For macro shots, MF helps you find the optimal focus plane without the hunting of auto focus.

---

---

That covers all five core parameters in smartphone Pro Mode. Understanding each one and how they interact will help you make confident adjustments for any shooting scenario.
